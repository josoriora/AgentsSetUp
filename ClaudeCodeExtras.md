
- Install JQ: https://jqlang.github.io/jq/download/
- You can customize your status line with /statusline command. you can use natural language and do manual config as well. 
    - The general details are in here:  https://code.claude.com/docs/en/statusline#set-up-a-status-line.
    - Here is a prompt you can use:
```
    Configure a global status bar for macOS that shows:
        - Model name in use
        - Context window percentage used
        - Tokens consumed / total tokens
        - Current directory
        - Git branch (if present)

    Create a separate .sh file for the script and save it in global settings.
```
    Obtained from here: https://pasqualepillitteri.it/en/news/162/claude-code-status-bar-context-monitor-guide
- This guide looks very useful: https://github.com/ykdojo/claude-code-tips
